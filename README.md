# Mouse_Disease_Detector
Mouse disease identification using an Android App.

Plan:
1. Use bing image downloader to collect images
2. Collect images for 10 diseases
3. Label Images
4. Train Classifier
5. Create Mobile App using Tensorflow Lite
6. Deploy
